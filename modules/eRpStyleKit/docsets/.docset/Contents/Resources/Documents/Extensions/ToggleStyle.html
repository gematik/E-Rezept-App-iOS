Available where Self == FormRadioToggleStyle

  • radio

    A toggle style that applies radio button like appearing as well as colors and paddings according to figma. Must be applied manually to toggles.

    To apply this style to a button, or to a view that contains buttons, use the View.buttonStyle(.radio) modifier.

ErxTaskCoreDataStore

Store for fetching, creating, updating or deleting ErxTasks and it‘s underlying types. Access to most entities is tied to the given profileId.

  • init(profileId:coreDataControllerFactory:foregroundQueue:backgroundQueue:dateProvider:)

    Initialize an ErxTask Core Data Store

  • fetchAuditEvent(by:)

    Fetch the ErxAuditEvent by its id when required by Self

  • fetchLatestTimestampForAuditEvents()

    Fetch the most recent timestamp of all AuditEvents

  • listAllAuditEvents(for:)

    List all audit events with the given local contained in the store

  • listAllAuditEvents(forTaskID:for:)

    Fetches all audit events related to a specific task id.

  • save(auditEvents:)

    Creates or updates a sequence of audit events into the store

  • fetchLatestTimestampForCommunications()

    Fetch the most recent timestamp of all Communications

  • listAllCommunications(for:)

    List all communications for the given profile contained in the store

  • allUnreadCommunications(for:)

    Returns all unread communications for the given profile

  • save(communications:)

    Creates or updates the passes sequence of ErxTaskCommunications

  • fetchChargeItem(by:fullDetail:)

    Fetch the ErxChargeItem by its id when required by Self

  • fetchLatestTimestampForChargeItems()

    Fetch the most recent enteredDate of all ChargeItems

  • listAllChargeItems()

    List all charge items with the given local contained in the store

  • save(chargeItems:)

    Creates or updates the passed sequence of ErxChargeItems

  • fetchTask(by:accessCode:)

    Fetch the entire database (incl. other profiles) for an ErxTask by its id and accessCode

  • fetchLatestLastModifiedForErxTasks()

    Fetch the most recent lastModified of all ErxTasks

  • listAllTasks()

    List all tasks contained in the store

  • listAllTasksWithoutProfile()

    List all tasks without relationship to a Profile

  • save(tasks:updateProfileLastAuthenticated:)

    Creates or updates a sequence of tasks into the store

  • delete(tasks:)

    Deletes a sequence of tasks from the store

  • listAllMedicationDispenses()

    List all medication dispenses contained in the store

  • save(medicationDispenses:)

    Creates or updates the passed sequence of ErxTask.MedicationDispenses

  • failing

    Initializes an instance of ErxTaskCoreDataStorewith a failing CoreDataController

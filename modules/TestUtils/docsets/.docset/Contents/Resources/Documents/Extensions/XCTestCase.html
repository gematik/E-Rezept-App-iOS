  • awaitPublisher(_:timeout:file:line:subscribeScheduler:receivingScheduler:)

    Convenient method to wait for the passed publisher to ether return one result or throw with an error. If you would like to receive more than one value call collectNext(_ count: Int) on the publisher you test in advance.

    Throws

    When no result has been received

ErxEpaMedication

public struct ErxEpaMedication : Hashable, Codable, Sendable

This structure represents a medication derived from the profile GEM_ERP_PR_Medication which by itself is derived from EPA_MEDICATION https://simplifier.net/epa-medication/epamedication For information on medication and it’s profiles go to: https://simplifier.net/packages/de.gematik.erezept-workflow.r4/1.4.3/files/2550130

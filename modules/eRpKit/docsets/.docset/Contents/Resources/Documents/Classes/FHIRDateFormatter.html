  • date(from:format:)

    Creates a Date from a string if it conforms to one of the FHIR date formats YYYY, YYYY-MM, YYYY-MM-DD or YYYY-MM-DDThh:mm:ss+zz:zz https://www.hl7.org/fhir/datatypes.html

    The method is idle to convert a date string from a server response since it used the ISO8601DateFormatter.

    Declaration

    Swift

    public func date(from string: String, format: DateFormats? = nil) -> Date?

    Parameters

    string

    a date string in one of the valid date FHIR formats or the one format defined explicit
    format

    the date format off the string which should be used to evaluate the string. Pass nil if the method should find the correct FHIR format.

    Return Value

    a Date constructed from the string if the format was correct, otherwise nil

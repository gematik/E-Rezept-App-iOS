    using document: DiscoveryDocument,
    redirect: String?
) -> AnyPublisher<IDPChallenge, IDPError>

    Parameters

    Parameters

    The unsigned IDP-Challenge. Generate a new challenge for each sso refresh by using requestChallenge.

    document
